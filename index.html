Glosario de Unidad 1 Lista de Palabras

  • A

    Sintaxis A

    Descripcion-.The a tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the a element is the href attribute, which indicates the link's destination.

  • ASIDE

    Sintaxis ASIDE

    Descripcion-. The aside tag defines some content aside from the content it is placed in. The aside content should be related to the surrounding content.

  • Audio

    Sintaxis Audio

    Descripcion-.The audio tag defines sound, such as music or other audio streams. Currently, there are 3 supported file formats for the audio element: MP3, Wav, and Ogg:

  • Body Tag Background

    Sintaxis Body Tag Background

    Descripcion-.The background attribute specifies a background image for a document.

  • Body

    Sintaxis Body

    Descripcion-.The body tag defines the document's body. The body element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

  • Button

    Sintaxis Button

    Descripcion-. The button tag defines a clickable button. Inside a button element you can put content, like text or images. This is the difference between this element and buttons created with the input element.

  • Br

    Sintaxis Br

    Descripcion-.The br tag inserts a single line break. The br tag is an empty tag which means that it has no end tag.

  • Console.log

    Sintaxis Console.log

    Descripcion-.JavaScript can "display" data in different ways: Writing into an HTML element, using innerHTML. Writing into the HTML output using document.write(). Writing into an alert box, using window.alert(). Writing into the browser console, using console.log().

  • Canvas

    Sintaxis Canvas

    Descripcion-.The canvas tag is used to draw graphics, on the fly, via scripting (usually JavaScript). The canvas tag is only a container for graphics, you must use a script to actually draw the graphics.

  • DIV

    Sintaxis Div

    Descripcion-. A div has no semantic meaning, but the section element does.

  • Drop-down list

    Sintaxis Drop-down list

    Descripcion-.Choose the appropriate tag to make a dropdown list?

    Select

  • DocumentElement

    Sintaxis DocumentElement

    Descripcion-.Returns the Element that is the root document element (for example, returns the html element in HTML documents).

  • The Document Object

    Sintaxis The Document Object

    Descripcion-.The document object is the root node of the HTML document and the "owner" of other nodes:(element nodes, text nodes, attribute nodes, and comment nodes). The document object provides properties and methods to access all node objects

  • Footer

    Sintaxis Footer

    Descripcion-.The footer tag defines a footer for a document or section. A footer element should contain information about its containing element.

  • Form

    Sintaxis Form

    Descripcion-.The form tag is used to create an HTML form for user input. The form element can contain one or more of the following form elements: input, textarea, button, select, option, optgroup, fieldset label

  • Input

    Sintaxis Input

    Descripcion-.The input tag specifies an input field where the user can enter data. input elements are used within a form element to declare input controls that allow users to input data. An input field can vary in many ways, depending on the type attribute.

  • Img

    Sintaxis Img

    Descripcion-.The img tag defines an image in an HTML page. The img tag has two required attributes: src and alt.

  • Link

    Sintaxis Link

    Descripcion-.The link tag defines a link between a document and an external resource. The link tag is used to link to external style sheets.

  • Li

    Sintaxis Li

    Descripcion-.The li tag defines a list item. The li tag is used in ordered lists(ol), unordered lists (ul), and in menu lists (menu).

  • Library Javascript
